# Manufacturing Strategy
---
The entire airframe was specifically designed for additive manufacturing using FDM 3D printing.

## Manufacturing Constraints
- Desktop printer build-volume limitations
- Lightweight structural requirements
- Modular assembly approach
- Reduced hardware count
- Simplified assembly workflow

## Manufacturing Solutions
- Segmented printable components
- 1 mm shell wall structure
- 20–30% gyroid infill
- Print-orientation optimisation
- Integrated alignment tabs and joints

## Materials Used
- PLA filament
- Wooden spars
- Structural epoxy adhesive
- M3 fasteners

---

## Engineering Challenges & Solutions

| Engineering Challenge | Solution |
|---|---|
| Printer bed size limitations | Segmented modular airframe with alignment features |
| Wing bending loads | Continuous spar reinforcement system |
| Torsional rigidity | Carry-through linkage structure |
| Maintenance accessibility | Screw-mounted removable hatches |
| Wiring management | Integrated cable-routing channels |
| Weight reduction | Thin-wall shell structure with gyroid infill |
| Assembly precision | Integrated locating and alignment features |

---

## Prototype Development

The project progressed from:
1. Conceptual aircraft sizing
2. Aerodynamic configuration development
3. Detailed CAD modelling
4. Structural optimisation
5. Additive manufacturing preparation
6. Prototype printing
7. Physical assembly
